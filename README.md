# sgs-Auto-Experience-Farming(三国杀自动刷战令脚本)
## Introduction
This project is aiming to farm the battle pass of the game Legends of The Tree kingdoms (sgs).
这个项目可以自动刷取三国杀每日任务，包括群英绘壁活跃度，战令经验，每日任务，元宝树等等。仅供技术学习使用，请勿用于盈利或破坏游戏环境
